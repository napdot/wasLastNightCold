# wasLastNightCold
Record a room temperature with Arduino.

## Components needed

- BreadBoard
- Arduino Nano
- DS18B20 1-Wire Temperature Sensor
- 4.7K Ohm Resistor
- HC - 05 Bluetooth Serial Module
